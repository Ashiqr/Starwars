# Starwars

# From graphql-swapi

This is an example of using GraphQL with the [Star Wars API (SWAPI)](https://swapi.dev/).

## Installation

```bash
yarn install
```

## Building Types

```bash
yarn generate
```

## Running Locally

```bash
yarn start
```

# Frontend

This Frontend to request and present the people from starwars

```bash
npm install
```

## Running Locally

npm start
